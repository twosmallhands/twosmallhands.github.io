##string中的+重载
每个加法运算符(+)的两侧的运算对象至少有一个是string
c++中的字符串字面值如”asdasd”并不是标准库类型string的对象，”asdasd”与string是不同类型

##string的插入
append()函数用来在字符串的末尾追加,返回string&

##string的赋值
assign()函数为先将原字符串清空，然后赋予新的值作替换,返回string&

##string的替换
replace()函数替换目标位置字符串，可以用insert()和erase()函数替代，但replace()函数简单

##string的插入
insert()函数用于在指定位置插入字符串
